# Проект о какающей сливками птичке.

Засыпаете сухие сливки, заливаете воду, а внешний пищевой тракт птички все переварит и выведет прямо у вашу кружку кофе.


## Инструкция

***Предупреджение**: я не профиссианальный инженер, это мой первый подобный проект.*

Для конструирования вам понадобятся*:
- микроконтроллер ардуино с установленной ардуино иде
- 3д принтер
- пайка с лотом
- клей-пистолет
- инструментарий типа кусачек, наждачной бумаги
- проводки с пинами
- ресиновые штуки для изоляции проводов //туду я хз как его звать
- 4 нпн транзистора
- 2 вращающихся ДС мотора
- 2 насоса на основе ДС моторов
- гнездо для батареек на 6.5В (минимум на 5В, максимум смотрите, чтобы детали не погорели)
- **время и нервы**, готовность тестировать и корректировать данные и распечатанные модельки из инструкции, если что-то не работает должным образом

* Конкретные модели, которые были применены мною, вы можете найти в ./other/README.md.

Найдите файл ./other/README.md. Изучите его, там находится детальная инструкция по сборке и тестированию.

Когда вы выполните пункты из ./other/README.md, можете приступать к коду. Клонируйте репозиторий и отредактируйте значения в файле ./Consts.h согласно вашей конфигурации. Далее, откройте файл ./oh_this_shitting_bird.ino в ардуино иде, скомпилируйте и загрузите на микроконтроллер.

Тестируем, при необходимости исправляем. Изменяйте исходный код, если это требуется.

Готово!
