# Проект о какающей сливками птичке.
(оч сырой репозиторий еще)
Засыпаете сухие сливки, заливаете воду, а внешний пищевой тракт птички все переварит и выведет прямо у вашу кружку кофе.

## Гайд по репозиторию
> ***Освобождение от ответственности**: я не профиссианальный инженер, это мой первый подобный проект.*
1. Инструкция по созданию физической модели располагается в файле **`./src/README.md`**.
2. Главный скрипт распологается в **`./main/main.cpp`**.
3. Протестируйте детали в **`./main/tests.h`** и внесите свои измерения в **`./main/Consts.h`** (калибровка).
