# Подготовка
## Материалы
Для конструирования вам понадобятся:

### Хардвейр:
- микроконтроллер ардуино
- 3д принтер
- пайка с лотом
- клей-пистолет
- инструментарий типа кусачек, наждачной бумаги
- проводки с пинами
- ресиновые трубочки для изоляции проводов
- 4 нпн транзистора
- 2 вращающихся ДС мотора
- 2 насоса на основе ДС моторов
- гнездо для батареек на 6.5В (минимум на 5В, максимум смотрите, чтобы детали не погорели)
- эпоксидный лак, безопасный для пищевых продуктов (крайне желательно, если вы собираетесь продукт от прибора потреблять)

### Софтвейр
- VS Code
- плагин PlatformIO
